# Платформа для организации и проведения мероприятий

Веб-приложение для размещения, управления и просмотра мероприятий,
разработанное в рамках учебной и производственной практик и представленное
на конкурс **IT-Планета 2025**.

Проект ориентирован на организации и пользователей, участвующих в мероприятиях,
и предназначен для упрощения процессов публикации событий, управления локациями
и взаимодействия с участниками.

---

Функциональность реализована в соответствии с техническим заданием конкурса IT-Планета 2025.
## Функционал:

**Пользователи и роли**
- Ролевая модель доступа: участник, организатор, модератор
- Аутентификация реализована на основе JWT
- Регистрация и авторизация пользователей
- Подача заявок на аккредитацию организаторов
- Модерация заявок на аккредитацию

**Мероприятия**
- Создание мероприятий организаторами
- Модерация и публикация мероприятий
- Просмотр списка мероприятий
- Интеграция с Яндекс Картами для отображения локаций

**Участие и взаимодействие**
- Участие пользователей в мероприятиях
- Добавление отзывов и оценок мероприятий
- Добавление других пользователей в друзья
- Приглашение друзей на мероприятия
